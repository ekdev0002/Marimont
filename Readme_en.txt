Information
-------------------------------------------------------------------------------
Marimont’s algorithm can detect the existence or not of a circuit in
a graph. This project is an implementation of this algorithm in C with the
GTK2 library.
     * Enter the list of vertices separated by semicolons (A; B; C)
     * Enter the list of arcs separated by semicolons ((A, B); (B, C); (C, A))

Running
-------------------------------------------------------------------------------
Unzip the Run.zip and launch the Marimont.exe

Coding
-------------------------------------------------------------------------------
